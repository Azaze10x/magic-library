# Magic Library — Landing Page

Static landing page for [magic-library.shamantech.co](https://magic-library.shamantech.co),
deployed via GitHub Pages (branch: `main`).

Magic Library is a desktop app where three AI agents — LORE, WRITER, and
CRITIC — collaborate to write your novel. Downloads ship as DMG assets on
this repo's [Releases](https://github.com/Azaze10x/magic-library/releases).
